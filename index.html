Профессии

Да, с нами ты станешь кем захочешь

Что ты освоишь:

  • Основы дизайна (композиция, типографика, цветовая теория)
  • Figma (авто-лейауты, компоненты, переменные, стили)
  • Прототипирование (кликабельный макет, юзер-флоу, Figma Mirror)
  • Адаптивный дизайн (сетки, брейкпоинты, авто-лейаут)
  • Работа с разработкой (DevMode, передача макета, Zeplin)
  • UX-исследования (персонажи, CJM, тестирование прототипов)
  • Портфолио (3 кейса: мобильное приложение + десктоп + лендинг)

Что ты освоишь:

  • HTML и CSS — семантика, флексы, гриды, адаптивная вёрстка
  • JavaScript — основы языка, DOM, события, работа с API
  • Современный фреймворк (React / Vue) и компонентный подход
  • Git и командная работа над проектом
  • Портфолио из реальных интерфейсов

Что ты освоишь:

  • Язык программирования (Python / Node.js / PHP)
  • Базы данных и SQL, проектирование схем
  • REST API и серверная логика
  • Аутентификация и безопасность приложений
  • Git, деплой и работа в команде

Что ты освоишь:

  • Управление проектами и постановка задач
  • Методологии Agile / Scrum / Kanban
  • Коммуникация с командой и заказчиком
  • Аналитика, метрики и планирование
  • Документация и управление рисками

Что ты освоишь:

  • Основы сетей: модель OSI, TCP/IP, маршрутизация
  • Настройка сетевого оборудования (коммутаторы, роутеры)
  • Администрирование серверов и служб
  • Сетевая безопасность и мониторинг
  • Диагностика и устранение неполадок
